# Contributing to CopSight AI

First off, thank you for considering contributing to CopSight AI. It's people like you that make CopSight AI such a great tool.

### 1. Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our Issues tab to see if someone else has already created a ticket. If not, go ahead and make one!

### 2. Fork & create a branch

If this is something you think you can fix, then fork CopSight AI and create a branch with a descriptive name.

### 3. Get the test suite running

Make sure your machine has the necessary dependencies installed (Node.js, Docker, Python). Run the test suite to ensure everything is working correctly before you begin your edits.

### 4. Implement your fix or feature

At this point, you're ready to make your changes. Feel free to ask for help; everyone is a beginner at first.

### 5. Code Quality standards

- Ensure all your code is formatted correctly.
- If you've added new functionality, please add corresponding tests.
- Please make sure all tests pass before submitting a Pull Request.

### 6. Make a Pull Request

At this point, you should switch back to your master branch and make sure it's up to date with CopSight AI's master branch. Then, push your feature branch and open a Pull Request.

Thank you!
